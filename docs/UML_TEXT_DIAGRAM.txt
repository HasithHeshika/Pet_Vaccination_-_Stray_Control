═══════════════════════════════════════════════════════════════════════════════
                              KEY FEATURES
═══════════════════════════════════════════════════════════════════════════════

🔐 AUTHENTICATION & SECURITY
   • Password hashing with bcrypt (pre-save hook)
   • JWT tokens with 7-day expiration
   • Role-based access (isAdmin flag)

🐾 PET MANAGEMENT
   • Auto-generated unique petId
   • QR code generation for each pet
   • Public pet profiles accessible via QR code
   • Lost & found tracking capability

💉 VACCINATION SYSTEM
   • Automated reminder scheduler (daily at 9 AM)
   • Email notifications 7 days before due date
   • Auto status updates (overdue detection)
   • Comprehensive vaccination history

📊 DATABASE INDEXES
   Vaccination model has optimized indexes:
   • pet + nextDueDate
   • nextDueDate + status
   • reminderSent + nextDueDate

🔔 NOTIFICATION SYSTEM
   • Email/SMS preference per vaccination
   • Automated reminder service
   • Tracks reminder sent status


═══════════════════════════════════════════════════════════════════════════════
                         API ENDPOINTS (Not in UML)
═══════════════════════════════════════════════════════════════════════════════

AUTH (/api/auth)
├── POST   /signup         Register new user
├── POST   /login          User login (returns JWT)
└── GET    /me             Get current user profile

USERS (/api/users) [Admin Protected]
├── GET    /               Get all users
├── GET    /:id            Get user by ID
├── PUT    /:id            Update user
└── DELETE /:id            Delete user

PETS (/api/pets)
├── POST   /               Register new pet [Auth Required]
├── GET    /               Get all pets [Admin Only]
├── GET    /user/:userId   Get user's pets [Auth Required]
├── GET    /:petId         Get pet by ID [Public]
├── PUT    /:petId         Update pet [Owner/Admin]
└── DELETE /:petId         Delete pet [Owner/Admin]

VACCINATIONS (/api/vaccinations)
├── POST   /               Create vaccination [Admin Only]
├── GET    /pet/:petId     Get pet vaccinations [Auth Required]
├── GET    /:id            Get vaccination by ID [Auth Required]
├── PUT    /:id            Update vaccination [Admin Only]
└── DELETE /:id            Delete vaccination [Admin Only]
